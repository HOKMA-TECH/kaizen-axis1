# Sketch Manifest

## Design Direction
Agenda minimalista e clean no estilo Linear/Notion. Tipografia simples, espaço em branco generoso, foco em legibilidade. A core action é "ver próximos eventos" — o layout deve priorizar uma lista/feed cronológica de compromissos, não uma grade de calendário pesada. Cores usadas apenas para categorização de tipo de evento (Visita, Reunião, Assinatura, Outro).

## Reference Points
- Linear (estrutura sidebar + feed de itens)
- Notion Calendar (split: calendário compacto + lista)
- Fantastical / Cron (mini-cal + event feed)
